# Складской учёт (лабораторная работа №4, вариант 7)

Веб-приложение для управления товарами, поставщиками и поставками на складе. Реализовано на FastAPI + SQLAlchemy, фронтенд – HTML/CSS/JS. Контейнеризация через Docker Compose.

## Функциональность
- CRUD товаров и поставщиков.
- Фиксация поставок (автоматическое увеличение остатков).
- Отчёты:
  - остатки на складе;
  - история поставок;
  - товары ниже минимального остатка.

## Запуск

### Локально (без Docker)
```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить backend (из корня проекта)
uvicorn backend.app.main:app --reload --host 0.0.0.0 --port 8010

# В другом терминале запустить frontend
cd frontend
python -m http.server 5500
